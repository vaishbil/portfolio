# 🖤 Vaishali Wankhede — Personal Portfolio
 
A dark-themed personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed and iterated from scratch — no frameworks, no templates.
 
**Live site → [vaishbil.github.io/portfolio](https://vaishbil.github.io/portfolio)**

---

## Tech Used
 
- HTML5, CSS3, JavaScript
- Google Fonts — Roboto + Fira Code
- Font Awesome icons
- Icons8 for skill icons
- Formspree for contact form submissions
- GitHub RSS feed + rss2json for live GitHub activity
- GitHub public API for live repo and star counts
- GitHub Pages for hosting

---

## Design
 
- Color palette — deep dark `#121212` background, `#bb86fc` purple accent, `#3700b3` deep purple
- Typography — Roboto for body, Fira Code for code/mono elements
- Sticky navbar with left purple accent bar
- Smooth scroll between sections
- Responsive for mobile
